### Method:
```sh
  - characterByDubling(STRING); ex: ahc04→ aahhcc004;
  - doublingTheString(STRING); ex: ahc04→ ahc04ahc04;
  - mirroringTheString(STRING); ex: ahc04→ 40cha;
  - determiningTheLength(STRING); ex: ahc04→ 5;
```
