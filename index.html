Installation

You need grunt and nodejs installed globally:

Fisrt download node.js after

$ npm install -g grunt-cli
$ git clone https://github.com/lupu60/bachelor_thesis.git
$ cd bachelor_thesis/car_server
$ npm install
$ grunt fisrt
$ npm start

acces localhost:3000
