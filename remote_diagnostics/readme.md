### Installation

You need grunt and nodejs installed globally:

Fisrt download [node.js] after
```sh
$ npm install -g grunt-cli
```

```sh
$ git clone https://github.com/lupu60/remote_diagnostics.git
$ cd remote_diagnostics
$ npm install
$ grunt production
$ npm start
```
